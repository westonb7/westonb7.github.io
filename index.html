Cancer Research

Master's Thesis

Analysis performed on breast cancer patients using publicly available data from The Cancer Genome Atlas for Master's thesis. Fellowship awarded from the Simmons Center for Cancer Research for this project, and the results were presented at an ISCB conference in Chicago. Analysis performed using data mining, machine learning, dimensionality reduction, performed using a supercomputing cluster. Data visualized using ggplot2 in R. Software for research written using Python, R, C++, and Bash.

  • Date: August 2017
  • Client: BYU
  • Category: Bioinformatics research

Mutational Signatures

Analysis Pipeline

Mutational signatures are quickly becoming a prominent point of research as biomarkers in the emerging field of personal medicine. Analysis pipeline developed for analyzing mutational signatures as a predictor for certain phenotypes. Project developed using Python, R, and Bash.

  • Date: February 2017
  • Client: BYU
  • Category: Bioinformatics

Heat Dispersion

Simulation

Heat dispersion model, simulating a heated CPU reaching thermodynamic equilibrium over a set number of iterations. Designed as a benchmarking test for a remote supercomputing cluster. Developed using C++, multithreading (OpenMP), C, and Bash.

  • Date: January 2016
  • Client: BYU
  • Category: Programming (class project)

Pseudomonas Research

Internship

Researched chimeric receptor proteins in pseudomonas at the Parales lab at UC Davis. Data collected by performing various enzyme assays on genetically altered pseudomonas. Data analysis performed using Numpy, Scipy, and MS Excel.

  • Date: April 2015
  • Client: UC Davis
  • Category: Biology
